# ImageViewer 图片查看器

一个基于 SwiftUI 开发的 macOS 图片查看器应用，支持浏览本地文件夹中的图片文件。

## 功能特性

### 图片浏览
- 选择本地文件夹加载图片
- 支持常见图片格式：JPG, JPEG, PNG, GIF, BMP, TIFF, WEBP, HIF, CR3
- 图片按文件名字母顺序排序显示

### 历史记录
- 自动保存最近打开的文件夹历史记录（最多20条）
- 支持从历史记录中快速打开文件夹
- 支持删除单个历史记录或清空全部历史记录
- 启动页默认显示最近文件夹列表

### 导航控制
- **键盘导航**：
  - 左箭头键：上一张图片
  - 右箭头键：下一张图片
  
- **界面按钮**：
  - 左右侧箭头按钮切换图片
  - "更换文件夹" 按钮更换图片文件夹
  - "时钟" 按钮查看历史记录

### 图片旋转
- **键盘控制**：
  - 上箭头键：逆时针旋转90度
  - 下箭头键：顺时针旋转90度
  - [ 键：逆时针旋转90度
  - ] 键：顺时针旋转90度
  - R 键：重置旋转

- **界面按钮**：
  - 旋转控制按钮组提供图形化操作

### 图片缩放
- **触控板手势**：
  - 双指捏合：缩放图片
  - 双指拖拽：移动图片（缩放后）

- **键盘快捷键**：
  - + 键：放大图片
  - - 键：缩小图片
  - Cmd + 0：重置缩放到100%

- **界面按钮**：
  - +/- 按钮控制缩放
  - "100%" 按钮重置缩放

### 图片删除
- **键盘控制**：
  - Delete 键：删除当前图片
  
- **操作流程**：
  - 按 Delete 键或在确认对话框中点击"移到废纸篓"
  - 文件将被移动到系统废纸篓而非直接删除
  - 支持 Tab 键在确认对话框按钮间切换

### 图片信息查看
- **界面按钮**：
  - "信息" 按钮查看当前图片详细信息
  - 显示文件名、大小、尺寸、创建时间和修改时间
  
- **操作方式**：
  - 支持 ESC 键关闭信息窗口

### 幻灯片播放
- **键盘快捷键**：
  - Cmd + P：开始/停止幻灯片播放
  
- **功能特性**：
  - 自动按3秒间隔播放图片
  - 播放完毕后自动停止

### 拖拽支持
- 支持将图片文件拖拽到应用窗口中打开

### 其他优化
- 图片预加载提升浏览体验
- 图片切换动画效果
- 完善的权限处理机制
- 错误处理和用户提示

## 系统要求
- macOS 10.15 或更高版本
- Xcode 12.0 或更高版本（用于编译）

## 使用方法
1. 启动应用
2. 点击 "选择文件夹" 按钮选择包含图片的文件夹
3. 使用键盘快捷键、触控板手势或界面按钮操作图片
4. 点击 "时钟" 按钮可以查看和管理历史记录

## 开发说明
- 项目使用 SwiftUI 框架构建
- 支持沙盒环境下的文件访问
- 遵循 macOS 人机界面指南设计
