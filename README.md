# ImageViewer 图片查看器

一个基于 SwiftUI 开发的 macOS 图片查看器应用，支持浏览本地文件夹中的图片文件。

## 功能特性

### 图片浏览
- 选择本地文件夹加载图片
- 支持常见图片格式：JPG, JPEG, PNG, GIF, BMP, TIFF, WEBP
- 图片按文件名字母顺序排序显示

### 历史记录
- 自动保存最近打开的文件夹历史记录（最多20条）
- 支持从历史记录中快速打开文件夹
- 支持删除单个历史记录或清空全部历史记录

### 导航控制
- **键盘导航**：
  - 左箭头键：上一张图片
  - 右箭头键：下一张图片
  
- **界面按钮**：
  - 左右侧箭头按钮切换图片
  - "Change Folder" 按钮更换图片文件夹
  - "History" 按钮查看历史记录

### 图片旋转
- **键盘控制**：
  - 上箭头键：逆时针旋转90度
  - 下箭头键：顺时针旋转90度
  - [ 键：逆时针旋转90度
  - ] 键：顺时针旋转90度
  - R 键：重置旋转

- **界面按钮**：
  - 旋转控制按钮组提供图形化操作

### 图片缩放
- **触控板手势**：
  - 双指捏合：缩放图片
  - 双指拖拽：移动图片（缩放后）

- **键盘快捷键**：
  - + 键：放大图片
  - - 键：缩小图片
  - Cmd + 0：重置缩放到100%

- **界面按钮**：
  - +/- 按钮控制缩放
  - "100%" 按钮重置缩放

## 系统要求
- macOS 10.15 或更高版本
- Xcode 12.0 或更高版本（用于编译）

## 使用方法
1. 启动应用
2. 点击 "Select Folder" 按钮选择包含图片的文件夹
3. 使用键盘快捷键、触控板手势或界面按钮操作图片
4. 点击 "History" 按钮可以查看和管理历史记录

## 开发说明
- 项目使用 SwiftUI 框架构建
- 支持沙盒环境下的文件访问
- 遵循 macOS 人机界面指南设计
